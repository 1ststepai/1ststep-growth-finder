# Cost Control

## Static/no-cost posture
- Keep this as a static page with no backend unless explicitly approved.
- Do not add paid scraping APIs, proxy services, enrichment services, or hosted automation.
- Browser localStorage and CSV export are sufficient for the current workflow.

## Outreach safety
- Search links should open public sites for manual review.
- Do not add auto-DM, auto-posting, or scraping.
- Do not add hidden analytics or tracking beyond explicit referral parameters.

## AI context hygiene
- Most changes should only require `AGENTS.md`, `README.md`, and the relevant section of `index.html`.
- Do not paste large CSV exports or prospect lists into Claude/Codex context.
- Keep referral/payout docs concise and easy to find.
